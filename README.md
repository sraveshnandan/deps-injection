# Express TypeScript Node Starter

A simple starter template for building applications with Express and TypeScript.

## Features

- TypeScript support
- Express framework
- Basic project structure
- Sample routes

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/sraveshnandan/deps-injection.git
   ```

2. Navigate to the project directory:

   ```bash
   cd deps-injection
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

To start the development server, run:

```bash
npm run dev
```

## Scripts

- `dev`: Starts the development server with hot reloading.
- `build`: Compiles TypeScript files to JavaScript.
- `start`: Runs the compiled JavaScript files.
